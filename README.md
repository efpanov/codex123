# Push Tracker Mini App

Мини-приложение для Telegram Web App, которое считает отжимания и хранит прогресс локально в браузере.

## Что уже есть

- установка цели по количеству отжиманий;
- главный экран с прогрессом в формате `сделано из цели`;
- кнопки `-10`, `-1`, `+1`, `+10`;
- сохранение цели и текущего количества в `localStorage`.

## Файлы

- `index.html` — разметка экрана;
- `styles.css` — стили интерфейса;
- `app.js` — логика счётчика и сохранения состояния.

## Как открыть

Открой `index.html` в браузере или подключи эти файлы к Telegram Web App как статический фронтенд.
